Process

STEP 1

원하는 컨설팅 선택

+

회원가입 후 Shop 페이지에서 원하는 VIBE PROFILE을 선택해주세요.
간단한 응답을 통해 수집된 데이터를 바탕으로 VIBE 분석 시스템이 개인의 분위기 패턴을 분류하고, 정리된 지표와 결과지를 제공합니다.

VIBE는 스타일을 추천하거나 바꾸라고 말하지 않습니다.
지금의 분위기가 어떤 결에 형성되어 있는지를 하나의 리포트로 정리해 보여줍니다.

VIBE PROFILE 구성

  • VIBE HAIR PROFILE : 헤어스타일 30종 분위기 범주 정리
  • VIBE MAKEUP PROFILE : 메이크업 형태 25종 분위기 범주 정리
  • VIBE FASHION PROFILE : 패션 아이템 35종 분위기 범주 정리
STEP 2

체크리스트 제출

+

구매 후 자정까지 응답을 완료해주세요.
제출된 응답은 VIBE 분석 시스템을 통해 개인의 분위기 패턴을 분류하고 결과지 생성에 사용됩니다.

응답 완료 후, 다음날 개인별 VIBE PROFILE 결과지가 제공됩니다.

STEP 3

분석 : MY VIBE REPORT 생성 과정

+

VIBE의 결과지는 사람의 판단이나 조언이 아닌, 응답 데이터를 기반으로 한 분석 시스템을 통해 생성됩니다.

STEP 1. DATA STRUCTURING

응답 데이터를 기준으로 형태·배치·흐름 정보를 구조화합니다.

STEP 2. PATTERN ANALYSIS

분위기 형성에 영향을 주는 주요 패턴을 분류합니다.

STEP 3. TYPE CLASSIFICATION

12가지 VIBE TYPE 중 해당 유형을 분류합니다.

STEP 4. PROFILE MAPPING

영역별 분위기 범주를 맵핑합니다.

STEP 5. REPORT GENERATION

개인별 결과지가 자동 생성됩니다.

VIBE는 지금의 나를 가장 명확하게 정리한 결과지 한 장을 만드는 시스템입니다.
